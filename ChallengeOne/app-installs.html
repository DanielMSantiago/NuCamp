Brave Browser | Spotify

How to Install the Brave Browser

  1. Go to Brave Browser Website and click on the "Download Brave" icon in the center of the screen.
  2. Once you click download and the download complete, you can click on the Brave Browser Install file to start the download.
  3. Once the download is completed, you will have a new icon for the brave browser.

How to Install Spotify

  1. You will need to create an account using Spotify Signup Page.
  2. Once you have created an account and selected your subscription service, you can go to your OS respected application store.
    • Microsoft: Microsoft Store
    • Apple: App Store
  3. Use the search bar to and search "spotify". Click on the install button. See logo below so you can easily identify the application
  4. Once installed, a pop up from the application stating to sign in or sign up. If you skipped step 1, the app will open a browser window to the sign up/in page.
  5. Once signed in to the profile, you will be able to access all the music and podcasts you like!
